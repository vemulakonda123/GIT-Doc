## "export PS1=" for customizing shell prompt

```
eg::::
hhv@12.32.65.98:/opt/hhv/jdjj$
export PS!='$'
$
```
## more command
```
more -n file1.txt
more -5 file1.txt
more -10 file1.txt
=======================
more +5 file1.txt >>>>>>>which show content from 5th line
```
## head and tail commands
```
head file.txt
head -10 file.txt
tail file.txt
tail -n file.txt
tail -10 file.txt
```
